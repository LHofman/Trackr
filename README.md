# Initialize project
-------------------
* Install Node
* Install npm
* Install Mongo

```shellsession
npm install
```

* Create a file '.env' in the root folder
```
DB_USERNAME='your_mongo_username'
DB_PASSWORD='your_mongo_password'
```

# start project
--------------
```shellsession
npm run dev
```
