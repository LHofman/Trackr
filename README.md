# Initialize project
-------------------
* Install Node
* Install npm
* Install Mongo

```shellsession
npm install
cd client
npm install
```

* Create a file '.env' in the root folder
```
DB_USERNAME='your_mongo_username'
DB_PASSWORD='your_mongo_password'
DB_SECRET='a_random_chosen_secret' //do not put 'a_random_chosen_secret', choose you own
```

# start project
--------------
```shellsession
npm run dev
```
